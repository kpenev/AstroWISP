Output a header to the given stream with version and command line information.

Includes the version of HATpipe if HATP_SVN is defined, the version of SubPixPhot if the version_string argument is not an empty string, and the command line with which the command was invoked if argc is non-zero, in which case argc and argv should be the same arguments that main() received.
