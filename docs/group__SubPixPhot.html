Detailed Description

Does aperture photometry correcting for non-uniform pixel sensitivity.

Uses knowledge about the source locations and PSFs on an image, as well as an externally provided map for the sensitivity of individual pixels (assumed all identical) to calculate the flux within a circular aperture around each source.

Properly accounts for pixels that stradle the aperture boundary and for the fact that some parts of the pixels are more sensitive than others.
