Create a grid according to a string.

Parameters
grid_stringThe string defining the grid. The format consists of comma separated list of x grid boundaries, optionally followed by ';' and a comma separated list of y grid boundaries. If the y grid boundaries part is omitted, the x_grid boundaries are used.
