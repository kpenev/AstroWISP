Detailed Description

template<typename DATA_TYPE>
class IO::TranslateToAny< DATA_TYPE >

Translator that works with boost::any for use with boost property trees.

If a value to translate is passed by reference, it is copied (good for small values). If instead a pointer is passed, the pointer is stored, but when the value is recovered the pointer is derereferenced. This avoids copying the object, but means that the object must not be destroyed (by e.g. going out of scope) before it is used.
