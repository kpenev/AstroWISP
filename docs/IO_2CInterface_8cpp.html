C-binding alias for IO::parse_hat_mask.

C-binding alias for IO::parse_hat_mask.

Parameters
mask_stringThe concatenated MASKINFO entries from the header.
x_resolutionThe horizontal resolution of the image.
y_resolutionThe vertical resolution of the image.
maskThe array to write the mask to. Only the masked entries are updated using logical or, thus calling this function with different mask_string values but the same mask array combines all masks.
