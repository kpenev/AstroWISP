Assert that the two values are equal to wihin the current tolerance.

Notes

The exact definition is:

|value_1 - value_2| <= tolerange * |value_1 + value_2| * float_info.epsilon

Parameters:

  • value_2 – The second of the two values to compare.

Returns:
