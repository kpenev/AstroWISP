Detailed Description

Measure the background in an annulus around each source.

Uses the pixels whose centers lie between two circles centered on each source to estimate the background (median) and its error (quantiles in the individual pixel values).

Pixels within the annulus for a given source are not included in the background determination if they fall inside the exclusion radius of another source.

measure the background for a source at (x,y).

Estimate the background around the given position as the median of the points whose centers lie within the annulus and not within the inner edge of another source's annulus.

The uncertainty of the determined value is estimated as half the size of the interval centered on the determined background value that contains the fraction of the points specified at construction as the error_confidence argument.

Implements Background::MeasureExcludingSources.
