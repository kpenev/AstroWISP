Wrap the given data in an image.

Parameters
valuesThe pixel values in the image. The x coordinate changes faster than the y with successive entries. The data is used directly rather than copying, so it must not be destroyed while this object is still in use and my be modified through this object. To force copying the data, use the constructor with const arguments.
maskQuality flags for the image pixels in the same order as values. See comments for values argument.
x_resolutionThe number of pixels along the x direction in the image.
y_resolutionThe number of pixels along the y direction in the image.
errorsError estimates of the entries in values. If NULL, any query to the error results in an exception.
