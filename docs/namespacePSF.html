Fill The PSF amplitudes in a data tree (computed if necessary).

If the amplitudes are not already available, both the PSF shape information should be and some form of flux information: fluxes or magnitudes + magnitude zero point. If fluxes are not available (as well as amplitudes) they will also be filled.
