Detailed Description

Calculates integrals of an elliptical gaussion over a rectangle.

Create an integral with the given S+D, S-D and K over a rectangle.

The rectangle to integrate over is given by x0-dx < x < x0+dx, y0-dy < y < y0+dy. Optionally the first (if calculate_first_deriv is true) and second (if calculate_second_deriv is true) derivatives of the integral are also calculated.
