Detailed Description

A base class for PSF models which are locally approximated by a polynomial of up to some degree defined at construction.

Integrates the PSF a wedge of a circle.

The wedge is defined by the following boundaries:

  • the line x=x
  • the line y=y
  • the circle centered at (0, 0) with a radius=radius If x is 0 the the left vs right wedge is chosen according to left Same for y0 and bottom.
Parameters
xThe left/right boundary of the wedge if the given value is positive/negative.
yThe bottom/top boundary of the wedge if the given value is positive/negative.
radiusThe radius of the circle giving the rounded boundary of the wedge.
leftIf x is exactly 0, this values is used to decide to which side of x the wedge extends.
bottomIf y is exactly 0, this values is used to decide to which side of y the wedge extends.

Implements PSF::PSF.
