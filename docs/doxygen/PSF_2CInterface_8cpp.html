Evaluate the given terms for the given list of sources.

Parameters
term_expressionsThe list of terms to evaluate. Usually created by expand_term_expression().
num_termsThe number of terms.
variable_namesThe names of the variables used by the expressions in terms.
variable_valuesThe values of the variables used by the expressions in terms for each source.
num_variablesThe number of different variables used.
num_sourcesThe number of sources to evaluate the terms of.
resultThe location to fill with the evaluated terms. Must already be allocated with a size of num_variables * num_sources. The values of all terms for the first source are at the beginning of the array, followed by the value of all terms for the second source etc.

Expand the given expression into a list of expressions, each defining a single PSF expansion term.

Parameters
expansion_term_expressionThe expression to parse. See TermGenerator::Grammar for a description of the syntax.
term_listThe individual terms in the expression. Allocates new memory. Caller must release allocated memory when no longer required.
num_termsOn exit, set to the number of terms found in the term expression.
