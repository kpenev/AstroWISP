Detailed Description

template<class PSF_TYPE>
class FitPSF::Source< PSF_TYPE >

An abstract base class for sources which participate in PSF fitting.

Creates a PSF fitting source.

Parameters
idThe ID of the source.
x0The x coordinate of the source center.
y0The y coordinate of the source center.
gainThe gain to assume for the backcground measurement.
backgroundThe background under the source (in ADU)
source_assignment_idThe id to assign to this source in the source_assignment array
subpix_mapThe sub-pixel sensitivity map to assume. Must not be destroyed while this object is in use.
image_idThe file name of the image from which this source was extracted.
output_fnameThe name of the file where this source should be saved after the fit.

template<class PSF_TYPE >
virtual void FitPSF::Source< PSF_TYPE >::pixel_excess_reductions ( const Eigen::VectorXd &  to_dot_with,
double &  dot_product,
double &  excess_sum_squares 
)
protectedpure virtual

Calculate the dot of pixel excesses with the given quantity as well as with itself.

Having this function virtual allows optimizing the calculation in some cases (e.g. when fitting for PSF which linearly depends on its parameters).

Parameters
to_dot_withA vector of values to dot with the pixel excess values.
dot_productOn exit this variable is set to the dot product of to_dot_with and the pixel excesses.
excess_sum_squaresOn exit this variable is set to the sum of the squares of the pixel excesses.

Implemented in FitPSF::OverlapSource< FIT_SOURCE_TYPE, PSF_TYPE >, FitPSF::OverlapSource< LinearSource, PSF::PiecewiseBicubic >, and FitPSF::LinearSource.
