Detailed Description

Avoid pixels close to sources when extracting a background.

A base class for background extractors which exclude some circular aperture around every source when determining the backgrounds of other sources.

Masks (assigns NaN to) the pixels in the given image whose centers lie within the specified aperture of at least one source. This masked array can then be used by a non-abstract child class to derive a background value and uncertainty.
