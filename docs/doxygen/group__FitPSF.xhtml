enum FitPSF::SourceDropReason

#include </Users/kpenev/projects/git/AstroWISP/doc_src/FitPSF/Common.h>

Define tags for reasons to exclude sources from the fit.

Enumerator
FEW_PIXELS 

Too few pixels were assigned to this source.

MANY_PIXELS 

Too many pixels were assigned to this source.

TOO_BIG 

Contains pixels too far from the source center.

OVERLAP 

This source overlaps with another.

NON_POINT_SOURCE 

This does not appear to be a point source.

BAD_BACKGROUND 

Failed to determine a reliable background.

PAST_MAX_SOURCES 

There is a sufficient number of "better" sources than this one.

MANY_SATURATED 

There were too many saturated pixels in the source.

NUM_DROP_REASONS 

How many reasons are there to drop sources.

NOT_DROPPED 

The source was not dropped.
