Fit the PSF of all sources simultaneously assuming smooth variation with position.

Perform a global fit for the polynomial expansions of S, D and K over the image using the given sources, starting from the given initial guess if supplied. Perform an individual fit to the PSFs of each source.

The starting point can optionally be specified. The arguments are identical to those of global_sdk_fit(). See there for a description of each argument. Fills an output data tree from SDK fit sources. Find the sources on which PSF fitting will be performed.

Parameters
optionsThe parsed command line options.
fit_sourcesGets filled with the sources to use for the shape fit.
dropped_sourcesGets fiiled with the sources dropped from the shape fit, still fit their amplitudes.
x_resolutionGets set to the common resolution of the input images in the x direction.
y_resolutionGets set to the common resolution of the input images in the y direction.
subpix_mapThe sub-pixel sensitivity map to assume.
psfA PSF that has grid that should be used for fitting.
output_data_treeThe tree to fill with output data.
